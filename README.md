# English

## Overview

**Voxelith** is a modern voxel editor built with Rust, featuring GPU-accelerated rendering via wgpu and a clean egui interface. It's designed as a procedural-first tool, meaning it prioritizes programmatic voxel generation alongside traditional manual editing.

### Core Features

- **Multi-tool Editing**: Place, Remove, Paint, Eyedropper, and Flood Fill tools
- **Undo/Redo**: Full command history with unlimited undo
- **Color Palette**: Customizable color palette for quick access
- **Viewport Controls**: Grid, axes display, and wireframe mode
- **Camera Presets**: Top, Front, Side views with smooth orbit controls

- **Sphere**: Gradient-colored sphere
- **Pyramid**: Egyptian-style stepped pyramid

## Installation

### Prerequisites

- Rust 1.70 or later
- A GPU with Vulkan, Metal, or DX12 support

### Building from Source

```bash
# Clone the repository
git clone https://github.com/Lynthar/Voxelith.git
cd Voxelith

# Build in release mode
cargo build --release

# Run
cargo run --release
```

### Module Structure

| Module | Description |
|--------|-------------|
| `core` | Voxel data structures (Voxel, Chunk, World) |
| `mesh` | Mesh generation from voxel data |
| `render` | GPU rendering with wgpu |
| `editor` | Tools, commands, and ray casting |
| `ui` | egui-based user interface |
| `io` | File import/export (VOX, VXLT) |
| `procgen` | Procedural generation utilities |

### Technical Specifications

- **Chunk Size**: 32Â³ voxels (32,768 per chunk)
- **Voxel Size**: 8 bytes (material, RGBA color, flags)
- **Rendering**: Naive meshing with per-face culling
- **Compression**: Gzip + RLE for project files
