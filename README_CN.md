## 概述

**Voxelith** 是一款使用 Rust 构建的现代体素编辑器，采用 wgpu 实现 GPU 加速渲染，配合简洁的 egui 界面。专为手动编辑和程序化生成设计。

## 功能特性

| 功能 | 说明 |
|------|------|
| 🎨 **多工具编辑** | 放置、删除、绘制、取色、填充 |
| ↩️ **撤销/重做** | 完整的命令历史 |
| 📁 **文件支持** | 原生 `.vxlt` 和 MagicaVoxel `.vox` |
| 🎲 **形状生成** | 立方体、球体、金字塔、地面 |
| 🖥️ **视口控制** | 网格、坐标轴、线框模式 |

## 快速开始

```bash
git clone https://github.com/Lynthar/Voxelith.git
cd Voxelith
cargo run --release
```

## 快捷键

| 按键 | 功能 | 按键 | 功能 |
|------|------|------|------|
| `1-5` | 选择工具 | `Ctrl+Z` | 撤销 |
| `WASD` | 移动相机 | `Ctrl+Y` | 重做 |
| `滚轮` | 缩放 | `Ctrl+S` | 保存 |
| `鼠标中键` | 轨道旋转 | `Ctrl+O` | 打开 |

## 技术栈

- 🦀 **Rust** - 系统级语言
- 🎮 **wgpu** - GPU 渲染
- 🖼️ **egui** - 即时模式 UI
- 🗜️ **flate2** - 压缩算法

## 架构设计

```
┌─────────────────────────────────┐
│        用户界面 (egui)           │
├─────────────────────────────────┤
│    编辑器 (工具, 命令)           │
├─────────────────────────────────┤
│   核心 (体素, 区块, 世界)        │
│   渲染 (wgpu) │ 文件 (vox, vxlt) │
└─────────────────────────────────┘
```
