## 概述

**Voxelith** 是一款使用 Rust 构建的现代体素编辑器，采用 wgpu 实现 GPU 加速渲染，配合简洁的 egui 界面。专为手动编辑和程序化生成设计。

## 功能特性

| 功能 | 说明 |
|------|------|
| 🎨 **编辑** | 放置 / 删除 / 绘制 / 取色 / 填充,支持拖动笔刷与笔画级 undo,实时悬停预览 |
| 🌱 **程序化生成** | Perlin 地形、L-System 树、WFC 地牢 —— 单生成器面板或可视化节点图编辑器组合 |
| ✨ **实时预览** | 防抖半透明叠加,生成结果落世界前可见 |
| 📁 **文件支持** | 原生 `.vxlt` 和 MagicaVoxel `.vox`(色彩有损量化时会提示) |
| 💾 **状态持久化** | 窗口布局、面板状态、生成器参数、最近文件跨重启保留 |
| 🖥️ **视口控制** | 轨道相机、网格、坐标轴、线框模式 |

## 快速开始

```bash
git clone https://github.com/Lynthar/Voxelith.git
cd Voxelith
cargo run --release
```

## 快捷键

| 按键 | 功能 | 按键 | 功能 |
|------|------|------|------|
| `1-5` | 选择工具 | `Ctrl+Z` | 撤销 |
| `WASD` | 移动相机 | `Ctrl+Y` | 重做 |
| `滚轮` | 缩放 | `Ctrl+S` | 保存 |
| `鼠标中键` | 轨道旋转 | `Ctrl+O` | 打开 |

## 技术栈

- 🦀 **Rust** - 系统级语言
- 🎮 **wgpu** - GPU 渲染
- 🖼️ **egui** - 即时模式 UI
- 🗜️ **flate2** - 压缩算法

## 架构设计

```
┌──────────────────────────────────────────────┐
│ UI(egui 面板 + 可视化节点图编辑器)         │
├──────────────────────────────────────────────┤
│ 编辑器(工具、命令、射线拾取、undo)         │
├──────────────────────────────────────────────┤
│ 程序化生成(地形 / 树 / WFC + DAG 求值)     │
├──────────────────────────────────────────────┤
│ 核心(体素、区块、世界)  │ 网格生成        │
│ 渲染(wgpu)              │ 文件 I/O 偏好    │
└──────────────────────────────────────────────┘
```
